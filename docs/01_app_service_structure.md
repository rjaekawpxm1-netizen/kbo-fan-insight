# KBO 앱 서비스 구조 분석 (작성용 템플릿)

> 공고 항목 "앱 메뉴·서비스 구조 분석", "기록·영상·굿즈·티켓 서비스 간 연계 구조 조사"에 대응.
> ⚠ 이 문서는 **실제 앱 사용·캡처가 있어야 정확히 작성**된다. 추측 기재 금지. 아래 틀을 채워라.

## [작성 방법]
1. KBO 공식 앱(스포츠투아이) 설치 → 주요 탭/메뉴를 캡처.
2. 메뉴 트리를 아래 형식으로 정리.
3. 기록·영상·굿즈·티켓이 서로 어떻게 연결(또는 단절)되는지 표시.

## 1. 메뉴/기능 트리 (채울 것)
```
홈
├─ (탭1)
├─ (탭2)
│  └─ (하위)
├─ 기록/순위
├─ 영상
├─ 티켓
└─ 굿즈/스토어
```
